Quick map of the utilities in this folder.

- `inspect_raw_file.py` — single entry point to summarize raw/ETH/structured files, decode waveforms, and (optional) raw→offline mapping.

- `read_daphne_eth.py`
- `daphne_eth_dump_and_summary.py`
- `daphne_channel_inventory.py`
- `dump_daphne_eth_stream_frame.py`

## Misc utilities
- `check.py`, `check_daphne_eth_summary.py`, `daphne_persistence_html.py`, `daphne_raw_offline_table.py` (deprecated; use `inspect_raw_file.py`), `debug_channel_env.py`, `header_dump.py`, `run_display.sh`, `get_rucio.py`, `fetch_rucio_replicas.py`, `setup_rucio_a9.sh`, `create_analysis_minimal_structure.sh`.

## Notes
- Prefer `inspect_raw_file.py` for new debugging instead of the older, single-purpose dump scripts.
- Structured outputs are generated by the updated readers (`04/06_RawHDF5Reader.py`, `01_Process.py`) using the unified loader.
